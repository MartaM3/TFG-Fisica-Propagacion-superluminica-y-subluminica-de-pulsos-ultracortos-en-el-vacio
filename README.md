# Superluminal and Subluminal Propagation of Ultrashort Pulses in Vacuum

This repository contains my Bachelor's Thesis in Physics and its accompanying materials.

The project studies the propagation of ultrashort optical pulses in vacuum, focusing on regimes in which the pulse peak exhibits superluminal or subluminal behaviour.

## Repository contents

- **Bachelor's thesis:** the complete dissertation, including the theoretical background, methodology, results and conclusions.
- **Python code:** scripts used to perform calculations and generate figures included in the thesis.
- **Technical appendix:** additional derivations and notes that expand on selected parts of the main text.

The code and supplementary document were originally prepared as supporting material. The full dissertation is also included so that the analysis and results can be read in context.

## Language

The thesis and supplementary material are written in Spanish. Variable names and comments in the Python scripts may also be in Spanish.

## Academic context

This work was submitted as a Bachelor's Thesis in Physics. The repository makes the dissertation and its supporting material available for academic and educational purposes.
